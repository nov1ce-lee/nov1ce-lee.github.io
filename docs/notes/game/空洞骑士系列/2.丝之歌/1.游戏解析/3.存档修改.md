想要查看存档是如何解析的内容，可以看[存档解析](2.存档解析.md)

找一个可以解析和修改空洞骑士存档的工具，这里是我之前借助ai开发的，可以直接使用
<LinkCard title="存档解析修改工具" icon="tabler:tools" href="https://nov1ce-lee.github.io/base64Tool/" />

导入存档文件后，即可得到解密后的JSON数据，可以找一个趁手的编辑器，把JSON数据贴进去修改

:::tip
玩家数据相关的主要就看playerData的内容，地图场景相关的就看sceneData的内容
:::
#### 主要配置
```json :::collapsed-lines=10
"maxHealthBase": 10,    // 最大生命
"silkMax": 18,          // 最大丝轴数
"geo": 5796,            // 念珠数/可以直接修改
"nailUpgrades": 4,      // 织针升级次数

"hasNeedleThrow": true,     // 丝之矛
"hasThreadSphere": true,    // 灵丝风暴
"hasParry": true,           // 十字绣
"hasHarpoonDash": true,     // 飞针冲刺
"hasSilkCharge": true,      // 丝刃镖
"hasSilkBomb": true,        // 符文之怒
"hasSilkBossNeedle": true,  // 苍白之爪
"hasNeedolin": true,        // 织忆弦针

"hasDash": true,            // 冲刺
"hasBrolly": true,          // 流浪者披风
"hasWalljump": true,        // 攀墙跳
"hasDoubleJump": true,      // 二段跳
"hasChargeSlash": true,     
"hasSuperJump": true,       // 超级
"hasQuill": true,           // 羽毛笔
"QuillState": 1,            // 羽毛笔类型，1：白色，2：红色，3：紫色
```
