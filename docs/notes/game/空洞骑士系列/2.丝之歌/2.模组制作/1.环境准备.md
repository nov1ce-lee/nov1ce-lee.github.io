### 游戏内环境配置
#### 1. BepInEx      
- BepInEx 是一个用于 Unity Mono、IL2CPP 和 .NET 框架游戏（XNA、FNA、MonoGame 等）的插件/模组框架，可以在游戏内加载插件        
    <LinkCard title="BepInEx下载" icon="mdi:github" href="https://github.com/BepInEx/BepInEx/releases" />

- 下载系统对应版本的BepInEx
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124161356698.png)
- 解压至游戏目录下
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124161703566.png)
- 在开发时可以修改`BepInEx\config\BepInEx.cfg`文件，将里面的日志打印开启方便调试
    ```ini
    [BepInEx]
    [Logging.Console]

    ## Enables showing a console for log output.
    # Setting type: Boolean
    # Default value: false
    Enabled = true
    ```

#### 2. UnityExplorer
- `UnityExplorer` 是一个用于 `Unity` 的 GUI 界面，用于查看和修改 `Unity` 的运行时数据，该工具可以辅助Mod开发
    <LinkCard title="UnityExplorer下载" icon="mdi:github" href="https://github.com/sinai-dev/UnityExplorer" />

- 下载BepInEx对应5.X版本的UnityExplorer
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124162934780.png)
- 解压至`BepInEx\plugins`目录下
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124163959516.png)

### 开发环境配置
#### 1. .NET SDK
- `.NET SDK` 是一个用于开发 `.NET` 应用程序的开发工具包，它提供了 `.NET` 的所有核心功能，如类库、编译器、调试工具等
    <LinkCard title=".NET SDK下载" icon="devicon:dot-net" href="https://dotnet.microsoft.com/zh-cn/download/dotnet" />

- 下载安装`.NET SDK` 8.0 或 9.0 版本
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124165356314.png)
- 安装完成后使用终端确认一下已安装的`.NET SDK`版本
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124183721598.png)
    ```sh
    dotnet --list-sdks
    dotnet --version    # 命令二选一即可
    ```

#### 2. Visual Studio Code
- `Visual Studio Code` 是一个开源的代码编辑器，好处是安装轻便
    <LinkCard title="Visual Studio Code下载" icon="logos:visual-studio-code" href="https://code.visualstudio.com/" />

- 安装`C#`插件即可使用，也可以使用`Visual Studio`进行开发      
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124184253191.png)

#### 3. Mod模板
- 这是一个用于开发《空洞骑士：丝之歌》 Mod 的模板，可以快速开始开发
    <LinkCard title="丝之歌Mod模板仓库" icon="mdi:github" href="https://github.com/silksong-modding/Silksong.Modding.Templates" />
- 打开终端，执行命令安装模板，该模板仅需要安装一次
    ```sh
    dotnet new install Silksong.Modding.Templates
    ```
    ![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124190046500.png)

- 打开你想创建模组所在的文件夹后，执行命令创建模板
    ```sh
    dotnet new silksongplugin --username YourGitHubUsername
    ```
    ![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124194303383.png)

### 反编译环境配置
#### 1. DnSpy

    <LinkCard title="DnSpy下载" icon="mdi:github" href="https://github.com/dnSpy/dnSpy/releases" />
- 下载对应版本并解压后使用即可
![](https://pic-1326566629.cos.ap-shanghai.myqcloud.com/notes/20260124194547948.png)
#### 2. FSMExpress
